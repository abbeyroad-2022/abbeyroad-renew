Project 06

GSC 셀프 ODT : 미래형 주유소 서비스 UI Planning, User Flow, Hi-Fi Prototyping, ProtoPie, Wireframing, UI Design, XD, Zeplin

전기차 충전소의 패러다임을 바꾸다

Energy Plus가 지향하는 방향과 고객에게 전달하고자 하는 이미지인 ‘Expandability, Proactive, Consumer Centric’를 모티브로 하여 브랜드 이미지를 강화 화고, 세련되고 현대적인 느낌을 전달하기 위해 고딕 서체와 그레이 색상을 사용하였으며, 불필요한 시각적 요소를 배제하여 복잡도가 최소화된 편리한 UI를 개발 하였습니다.

Flow of Use

주유 단계 단축으로 사용자의 수행 시간을 줄여 효율성을 극대화하였으며 익숙하고 간결한 UI를 통해 사용자가 키오스크 조작에 어려움이 없도록 하였습니다.

Payment

결제수단에 관계없이 사용자에게 일관된 경험을 제공하기 위해 다양한 결제서비스를 도입하였습니다.

Handy features

바코드 or 주유번호 입력만으로 주유를 간편하게 할 수 있는 ‘ 바로주유’ 기능을 도입하여 gs포인트 적립 및 할인쿠폰 사용에 결제까지 한번에 진행이 가능해 좀 더 빠르고 편리하게 서비스를 이용할 수 있습니다.

Refueling

주유 중 단계별 화면을 통해 현재 진행 상태를 인지할 수 있으며 각 화면에 따른 행동 안내를 통해 사용자의 동작을 유도하였습니다.
