Project 06

Ocha : AI 크로스 검색(Cross Search)을 통한 맛집 추천 플랫폼 UI Planning, User Flow, Hi-Fi Prototyping, ProtoPie, Wireframing, UI Design, XD, Framer, Zeplin

Ocha

빅데이터를 통해 위치를 기반으로 음식점에 대한 다양한 정보를 구조화하고 단순히 검색의 나열이 아닌 교차검색과 비슷한 취향의 사람들이 남긴 리뷰평점을 활용하여 다양한 추천이 가능하도록 함으로써 정보취득의 단계를 최소화시키고 만족스러운 정보를 제공받을 수 있도록 설계하였습니다.

Big Data Match-Up

음식점 POI 데이터를 활용하여 상용자들의 위치를 기반으로 빠르게 검색함과 동시에 크로스 검색을 통한 취향에 근접한 음식점을 추천할 수 있는 시스템을 구현하며 검색 결과와 추천을 이질감없이 배열 할 수 있는 UI를 설계하였습니다.

Recommendation System

카테고리별 비슷한 취향의 그룹을 구성함으로써 팔로잉하여 서로 정보를 공유할 수 있으며 리뷰평점을 높게 부여한 식당을 데이터화하여 비슷한 취향의 사람들에게 추천 코스로 제시하는 시스템을 도입하였습니다. 또한, 팔로잉 포인트를 통하여 해당 음식점에서 할인받을 수 있게함으로써 팔로잉 활성화를 동기부여할 수 있게 구조화시켰습니다.
